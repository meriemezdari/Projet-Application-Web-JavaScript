# Projet – Application Web JavaScript

**Présentation :** Jeudi 22 janvier 2026

**Projet réalisé en collaboration avec :** Nosayba EL JEBLI

Vous pouvez tester l'application en ligne ici : [Gestion de livres](https://meriemezdari.github.io/Projet-Application-Web-JavaScript/)

---

## Objectif du projet
Développer une mini-application web dynamique en JavaScript.  
L'objectif est de mobiliser toutes les compétences acquises depuis le début du cours :  
manipulation du DOM, structures de contrôle, tableaux, objets, gestion des événements, et persistance locale.

---

## Consignes générales
- Projet à réaliser en binôme.
- Présentation lors de la dernière séance : Jeudi 22 janvier 2026.
- L'application doit fonctionner **sans serveur** (HTML + CSS + JavaScript uniquement).
- Code clair, commenté et organisé.
- Interface conviviale attendue.

---

## Choix du sujet (au choix)
Vous pouvez choisir parmi les sujets suivants ou proposer une idée personnalisée (avec validation préalable) :

### 1. Gestionnaire de tâches (todolist)
- Ajout, modification et suppression de tâches.
- Chaque tâche a un nom, une date limite, un statut (fait / à faire).
- Sauvegarde dans `localStorage`.
- Tri par date ou par statut.

### 2. Gestionnaire de notes d'étudiants
- Ajouter des étudiants avec nom, prénom, et notes.
- Calcul automatique de la moyenne.
- Affichage dynamique dans un tableau.
- Filtrage des étudiants selon leur moyenne.

### 3. Application de gestion de livres
- Ajouter un livre (titre, auteur, année, genre).
- Afficher la liste des livres.
- Rechercher un livre par titre ou auteur.
- Trier les livres par année ou nom d'auteur.

## Conseils
- Commencez par une maquette papier ou un wireframe.
- Divisez le travail dans le binôme (DOM vs logique JS, par exemple).
- Testez régulièrement votre code.
- Sauvegardez vos fichiers dans un répertoire unique.
